# 它的未来
- 规范方便拓展的指令集

它可能和 语法处理器结合,来使汇编不再依靠手工生成.结合C4项目.
- syscall
它可能和 SuperShell结合,使它具有调用系统函数的本领.

它的某个分支可能加入混淆功能.
- 即时编译-巅峰
它可能增加即时编译功能,使它能够编译成Native代码提高效率.
